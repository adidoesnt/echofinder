# EchoFinder Telegram Bot

This is the EchoFinder Telegram bot. It allows you to search for messages, and returns results based on similarity.

## Setup

### Dependencies

To install dependencies:

```bash
bun install
```

### Environment Variables

```
TELEGRAM_BOT_TOKEN=
NODE_ENV=
BACKEND_HOST=
BACKEND_PORT=
API_CLIENT_TIMEOUT=
BACKEND_API_KEY=
PLATFORM=
```

### Running

To run:

```bash
bun run start

# or

bun run dev
```
