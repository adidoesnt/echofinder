# Health Check

## Setup

### Dependencies

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run start

# or

bun run dev
```

### Environment Variables

```bash
SERVER_LIST=
PING_INTERVAL=
LOG_FILE=
ERROR_LOG_FILE=
```

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
